Task 0: Βάση Δεδομένων & Μοντελοποίηση Δεδομένων

Πριν γράψουμε κώδικα σε Python, πρέπει να ορίσουμε πώς θα αποθηκεύονται τα δεδομένα.

Εργασία 1: Σχεδιασμός Σχήματος SQLite. Δημιουργήστε πίνακες για Συνταγές, Υλικά και Βήματα. Βεβαιωθείτε ότι ο πίνακας Βημάτων περιλαμβάνει πεδία για τίτλο, περιγραφή, διάρκεια_ώρες και διάρκεια_λεπτά.

Εργασία 2: Δομή Κλάσεων Python (Αντικειμενοστραφής Προγραμματισμός - OOP). Ορίστε τις κλάσεις σας στην Python (π.χ. Recipe, Step, Ingredient) ώστε να αντιστοιχούν στους πίνακες της βάσης δεδομένων.
