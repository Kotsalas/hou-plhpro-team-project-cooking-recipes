# Σύστημα Διαχείρισης Συνταγών Μαγειρικής (Project 50)

Αυτό το project αναπτύσσεται στο πλαίσιο της εργασίας για το **Ελληνικό Ανοικτό Πανεπιστήμιο (ΕΑΠ)**.

## 📝 Περιγραφή
Η εφαρμογή επιτρέπει στους χρήστες να:
* Καταχωρούν νέες συνταγές (όνομα, κατηγορία, δυσκολία, υλικά, βήματα).
* Αναζητούν συνταγές βάσει ονόματος ή κατηγορίας.
* Τροποποιούν ή διαγράφουν υπάρχουσες συνταγές.
* Εκτελούν μια συνταγή βήμα-βήμα με υπολογισμό προόδου (%).

## 🛠️ Τεχνολογίες
* **Γλώσσα:** Python 3
* **Βάση Δεδομένων:** SQLite3
* **Γραφικό Περιβάλλον (GUI):** Tkinter
