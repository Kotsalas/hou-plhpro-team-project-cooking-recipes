# Σύστημα Διαχείρισης Συνταγών Μαγειρικής (Project 50)

Αυτό το project αναπτύσσεται στο πλαίσιο της εργασίας για το **Ελληνικό Ανοικτό Πανεπιστήμιο (ΕΑΠ)**.

## 📝 Περιγραφή
Η εφαρμογή επιτρέπει στους χρήστες να:
* Καταχωρούν νέες συνταγές (όνομα, κατηγορία, δυσκολία, υλικά, βήματα).
* Αναζητούν συνταγές βάσει ονόματος ή κατηγορίας.
* Τροποποιούν ή διαγράφουν υπάρχουσες συνταγές.
* Εκτελούν μια συνταγή βήμα-βήμα με υπολογισμό προόδου (%).

## 🛠️ Τεχνολογίες
* **Γλώσσα:** Python 3
* **Βάση Δεδομένων:** SQLite3
* **Γραφικό Περιβάλλον (GUI):** Tkinter

## Ρύθμιση Βάσης Δεδομένων

Η βάση δεδομένων του συστήματος υλοποιείται με SQLite.

Η δημιουργία των πινάκων γίνεται με το αρχείο:

backend/database.py

Για την εισαγωγή ενδεικτικών δεδομένων χρησιμοποιείται το script:

backend/seed.py

## Εκτέλεση Εφαρμογής

### Βήμα 1: Δημιουργία Βάσης Δεδομένων

Πρώτα, δημιουργήστε τη βάση δεδομένων:

```bash
python backend/database.py
```

### Βήμα 2: Προσθήκη Ενδεικτικών Δεδομένων (Προαιρετικό)

Για να προστεθούν δοκιμαστικές συνταγές:

```bash
python backend/seed.py
```

### Βήμα 3: Εκτέλεση της Εφαρμογής

**Με Γραφικό Περιβάλλον (GUI) - Συνιστάται:**

```bash
python gui/app_gui.py
```

Η εφαρμογή GUI προσφέρει:
- Εύκολη πλοήγηση με κουμπιά και μενού
- Οπτική προβολή συνταγών σε πίνακα
- Φόρμες για δημιουργία και επεξεργασία
- Progress bar για εκτέλεση συνταγών
- Υποστήριξη scrolling με mouse/touchpad

**Με Γραμμή Εντολών (CLI):**

```bash
python backend/app_cli.py
```

## Σημείωση

Το αρχείο `recipes.db` δημιουργείται τοπικά μετά την εκτέλεση του `database.py` και δεν αποθηκεύεται στο GitHub.

## Τελευταίες αλλαγές

- Προστέθηκε πλήρης γραφική διεπαφή (GUI) με Tkinter.
- Υποστήριξη scrolling με mouse wheel και touchpad.
- Προστέθηκε validation στο CLI για ώρες, λεπτά και ID.
- Τα βήματα και τα υλικά εμφανίζονται με πιο καθαρά μηνύματα.
- Ενημερωμένα σχόλια στο service (`recipes_service.py`) για καλύτερη κατανόηση του κώδικα.

## Λειτουργίες

### GUI (Γραφικό Περιβάλλον)

Η εφαρμογή GUI υποστηρίζει:

1. **Αναζήτηση συνταγών** - Αναζήτηση με όνομα ή κατηγορία
2. **Δημιουργία νέας συνταγής** - Πλήρης καταχώρηση με υλικά και βήματα
3. **Προβολή λεπτομερειών** - Εμφάνιση όλων των στοιχείων συνταγής
4. **Τροποποίηση** - Ενημέρωση βασικών στοιχείων συνταγής
5. **Εκτέλεση βήμα-βήμα** - Πλοήγηση με progress bar και υπολογισμό προόδου (%)
6. **Διαγραφή συνταγής** - Με επιβεβαίωση

### CLI (Γραμμή Εντολών)

Η εφαρμογή CLI υποστηρίζει:

1. Δημιουργία νέας συνταγής  
2. Αναζήτηση συνταγής  
3. Προβολή λεπτομερειών συνταγής  
4. Εκτέλεση συνταγής βήμα-βήμα  
5. Τροποποίηση ή διαγραφή συνταγής
