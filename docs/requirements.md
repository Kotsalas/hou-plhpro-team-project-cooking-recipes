#### Τίτλος Project: Ανάπτυξη εφαρμογής διαχείρισης συνταγών μαγειρικής
#### ID Project: 50

---

Το υπό ανάπτυξη σύστημα είναι ένα σύστημα καταγραφής συνταγών μαγειρικής. Το σύστημα θα πρέπει να υποστηρίζεται από μία βάση δεδομένων στην οποία θα αποθηκεύονται πληροφορίες για τις συνταγές και τα υλικά τους, τους τύπους συνταγών, τα βήματα εκτέλεσης μιας συνταγής με τους χρόνους εκτέλεσης κάθε βήματος. Το σύστημά προαιρετικά μπορεί να είναι αντικειμενοστρεφές με τις κατάλληλες κλάσεις. Χρησιμοποιείστε για την βάση δεδομένων την sqlite3. Επίσης δημιουργήστε την γραφική διασύνδεση χρήστη με την βιβλιοθήκη Tkinter.

<b><u>Ζητούμενα:</u></b>

Να γράψετε κώδικα στη γλώσσα προγραμματισμού python ο οποίος θα υποστηρίζει τις ακόλουθες περιπτώσεις χρήσης:
1. <b>Καταχώρηση συνταγής:</b> Σ’ αυτή την περίπτωση χρήσης ο χρήστης θα πρέπει να μπορεί να κάνει καταχώρηση συνταγής χρησιμοποιώντας το σύστημα. Ο χρήστης θα πρέπει να καταχωρήσει (1) το όνομα της συνταγής (π.χ. καρμπονάρα), (2) την κατηγορία της συνταγής (π.χ. ζυμαρικά), (3) βαθμό δυσκολίας (δύσκολη, μέτρια, εύκολη),  συνολικό χρόνο εκτέλεσης (π.χ. 20 λεπτά), (4) τα υλικά της συνταγής, (5) τα βήματα εκτέλεσης της συνταγής. Σε σχέση με τα βήματα εκτέλεσης θα πρέπει να είναι διακριτές οντότητες και να καταχωρούνται ένα-ένα.  Κάθε βήμα θα έχει ένα τίτλο, μία περιγραφή (π.χ. βράζουμε τα μακαρόνια), τα υλικά που χρειάζονται για το βήμα καθώς και χρονική διάρκεια του βήματος σε ώρες και λεπτά (π.χ. 0 ώρες και 10 λεπτά). Τα βήματα θα πρέπει να είναι οργανωμένα σε ακολουθία (1ο, 2ο, 3ο κ.λπ.)
2. <b>Αναζήτηση συνταγής:</b> Σ’ αυτή την περίπτωση χρήσης ο χρήστης θα πρέπει να μπορεί να αναζητήσει μία συνταγή με βάση το όνομά της ή την κατηγορία της. Το σύστημα θα εμφανίζει όλες τις συνταγές που ταιριάζουν στην αναζήτηση ή μήνυμα πως δεν βρέθηκε καμία συνταγή. Από την οθόνη του αποτελέσματος της αναζήτησης, ο χρήστης θα μπορεί να διαγράψει/τροποποιήσει/εκτελέσει μία συνταγή αφού την επιλέξει.

2. <b>Τροποποίηση/διαγραφή της συνταγής:</b> Σε αυτή την περίπτωση χρήσης θα πρέπει να μπορεί ο χρήστης να τροποποιήσει όλα τα στοιχεία μιας συνταγής, συμπεριλαμβανομένων και των βημάτων εκτέλεσης αλλά και να διαγράψει την συνταγή συνολικά. Επίσης θα πρέπει να είναι δυνατή η διαγραφή μεμονωμένων βημάτων της συνταγής. 
3. <b>Εκτέλεση συνταγής:</b> Σε αυτή την περίπτωση χρήσης, ο χρήστης θα πρέπει να μπορεί να υποβοηθείται από το σύστημα κατά την εκτέλεση μιας συνταγής. Ο χρήστης θα ξεκινά την εκτέλεση της συνταγής και τότε θα περνάει στο πρώτο βήμα. Θα εμφανίζονται στην οθόνη τα στοιχεία του πρώτου βήματος (τίτλος, υλικά, χρόνος) καθώς και το ποσοστό της συνταγής που έχει ολοκληρωθεί (αρχικά μηδέν). Για το ποσοστό εμφάνισης θα πρέπει να λαμβάνεται υπόψη ο χρόνος των βημάτων σε σχέση με τον συνολικό χρόνο της συνταγής και όχι απλώς η αρίθμηση των βημάτων. Ο χρήστης θα μπορεί να περάσει στο επόμενο βήμα κ.ο.κ. Αυτό θα γίνεται έως ότου ολοκληρωθούν όλα τα βήματα της συνταγής.

Μπορείτε προαιρετικά να κάνετε χρήση του αντικειμενοστρεφούς παραδείγματος ανάπτυξης. Σ’ αυτή την περίπτωση θα πρέπει να ορίσετε κατάλληλες κλάσεις για την Συνταγή, τα Υλικά, τα Βήματα και ότι άλλο κρίνετε απαραίτητο. Οι κλάσεις αυτές θα πρέπει να συσχετίζονται μεταξύ τους. Έτσι για παράδειγμα μία συνταγή θα διατηρεί μία λίστα με υλικά και μία άλλη με τα βήματα της συνταγής, κάνοντας έτσι εύκολη την πλοήγηση από την μία κλάση στην άλλη. Όταν για παράδειγμα έχουμε μία συνταγή είναι εύκολο να παρουσιάσουμε όλες τις λεπτομέρειες των υλικών της και των βημάτων της.

Αντίστοιχα η βάση δεδομένων θα πρέπει να είναι σχεσιακή και να έχει πίνακες με κατάλληλα πεδία για τα δεδομένα της εφαρμογής. Σε περίπτωση εφαρμογής του αντικειμενοστραφούς παραδείγματος ανάπτυξης, θα πρέπει να αντιστοιχίσετε το σχεσιακό μοντέλο της βάσης δεδομένων στον αντικειμενοστρεφές μοντέλο του προγράμματος. Για παράδειγμα αν έχετε στην βάση δεδομένων έναν πίνακα Recipes για τις συνταγές θα έχετε αντίστοιχα και μία κλάση Recipe στο πρόγραμμά σας.

Επίσης προαιρετικά μπορείτε να χρησιμοποιήσετε εικόνες για να ωραιοποιήσετε την εφαρμογή σας (π.χ. μπορείτε να εισάγετε φωτογραφίες της συνταγής).
