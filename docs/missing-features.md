# Λείπουσες Λειτουργίες - Κρίσιμα Ζητήματα

## Περίληψη
Αυτό το έγγραφο καταγράφει τις 3 κρίσιμες λειτουργίες που λείπουν από την τρέχουσα υλοποίηση του συστήματος διαχείρισης συνταγών, σύμφωνα με τις απαιτήσεις του Project 50.

---

## 1. Τροποποίηση και Διαγραφή Βημάτων

### Απαίτηση από requirements.md
> "Σε αυτή την περίπτωση χρήσης θα πρέπει να μπορεί ο χρήστης να τροποποιήσει όλα τα στοιχεία μιας συνταγής, συμπεριλαμβανομένων και των βημάτων εκτέλεσης αλλά και να διαγράψει την συνταγή συνολικά. **Επίσης θα πρέπει να είναι δυνατή η διαγραφή μεμονωμένων βημάτων της συνταγής.**"

### Τρέχουσα Κατάσταση
- ✅ Υπάρχει `add_step()` για προσθήκη βημάτων
- ❌ Δεν υπάρχει `update_step()` για τροποποίηση βημάτων
- ❌ Δεν υπάρχει `delete_step()` για διαγραφή βημάτων
- ❌ Το GUI (`recipe_details_window.py`) εμφανίζει τα βήματα μόνο για ανάγνωση (read-only)

```python
def update_step(step_id: int, title: str, description: str, duration_minutes: int) -> bool:

    pass

def delete_step(step_id: int) -> bool:
    """Διαγράφει ένα βήμα και αναδιοργανώνει τη σειρά των υπόλοιπων"""
    pass
```

### Απαιτούμενες Αλλαγές
1. **Backend**: Προσθήκη συναρτήσεων `update_step()` και `delete_step()` στο `recipes_service.py`
2. **GUI**: Τροποποίηση του `recipe_details_window.py` για να επιτρέπει:
   - Επεξεργασία τίτλου, περιγραφής και χρόνου κάθε βήματος
   - Κουμπί διαγραφής για κάθε βήμα
   - Αναδιοργάνωση σειράς βημάτων (προαιρετικά)

## 2. Διαχείριση Υλικών (Προσθήκη/Αφαίρεση)

### Απαίτηση από requirements.md
> "Σε αυτή την περίπτωση χρήσης θα πρέπει να μπορεί ο χρήστης να τροποποιήσει **όλα τα στοιχεία μιας συνταγής**, συμπεριλαμβανομένων και των βημάτων εκτέλεσης"

Τα υλικά είναι μέρος των στοιχείων της συνταγής και των βημάτων.

### Τρέχουσα Κατάσταση
- ✅ Υπάρχει `add_ingredient_to_recipe()` για προσθήκη υλικών στη συνταγή
- ✅ Υπάρχει `add_ingredient_to_step()` για προσθήκη υλικών σε βήμα
- ❌ Δεν υπάρχει τρόπος αφαίρεσης υλικών από συνταγή
- ❌ Δεν υπάρχει τρόπος αφαίρεσης υλικών από βήμα
- ❌ Το GUI δεν επιτρέπει καμία τροποποίηση υλικών

### Λείπουσες Συναρτήσεις στο `recipes_service.py`

```python
def remove_ingredient_from_recipe(recipe_id: int, ingredient_name: str) -> bool:
    """Αφαιρεί υλικό από τη συνταγή"""
    pass

def remove_ingredient_from_step(step_id: int, ingredient_name: str) -> bool:
    """Αφαιρεί υλικό από συγκεκριμένο βήμα"""
    pass
```

### Απαιτούμενες Αλλαγές
1. **Backend**: Προσθήκη συναρτήσεων αφαίρεσης υλικών
2. **GUI**: Προσθήκη διεπαφής για:
   - Προσθήκη νέων υλικών στη συνταγή
   - Αφαίρεση υλικών από τη συνταγή
   - Προσθήκη/αφαίρεση υλικών από κάθε βήμα

## 3. Πλήρης Τροποποίηση Συνταγής μέσω GUI

### Απαίτηση από requirements.md
> "**Τροποποίηση/διαγραφή της συνταγής:** Σε αυτή την περίπτωση χρήσης θα πρέπει να μπορεί ο χρήστης να τροποποιήσει όλα τα στοιχεία μιας συνταγής, συμπεριλαμβανομένων και των βημάτων εκτέλεσης"

### Τρέχουσα Κατάσταση στο GUI

**Αρχείο**: `gui/recipe_details_window.py`

Τρέχουσα λειτουργικότητα:
- ✅ Τροποποίηση βασικών στοιχείων (όνομα, κατηγορία, δυσκολία, χρόνος)
- ❌ Δεν επιτρέπει τροποποίηση υλικών συνταγής
- ❌ Δεν επιτρέπει τροποποίηση βημάτων
- ❌ Δεν επιτρέπει τροποποίηση υλικών βημάτων

### Τρέχων Κώδικας (Γραμμές 107-115)
```python
# Υλικά συνταγής
ttk.Label(main_frame, text="Υλικά Συνταγής", 
         font=("Arial", 12, "bold")).grid(row=7, column=0, columnspan=2, pady=(20, 10))

ingredients_text = tk.Text(main_frame, width=50, height=6, state='disabled')
ingredients_text.grid(row=8, column=0, columnspan=2, pady=5)
```

**Πρόβλημα**: Το Text widget είναι `state='disabled'` - μόνο για ανάγνωση.

### Τρέχων Κώδικας (Γραμμές 122-138)
```python
# Βήματα
ttk.Label(main_frame, text="Βήματα Εκτέλεσης", 
         font=("Arial", 12, "bold")).grid(row=9, column=0, columnspan=2, pady=(20, 10))

steps_text = tk.Text(main_frame, width=50, height=15, state='disabled')
steps_text.grid(row=10, column=0, columnspan=2, pady=5)
```

**Πρόβλημα**: Τα βήματα εμφανίζονται σε Text widget μόνο για ανάγνωση.

### Απαιτούμενες Αλλαγές στο GUI

1. **Διαχείριση Υλικών Συνταγής**:
   - Listbox με τα τρέχοντα υλικά
   - Entry + κουμπί "Προσθήκη Υλικού"
   - Κουμπί "Αφαίρεση Επιλεγμένου Υλικού"

2. **Διαχείριση Βημάτων**:
   - Listbox ή Treeview με τα βήματα
   - Κουμπί "Επεξεργασία Βήματος" → άνοιγμα dialog
   - Κουμπί "Διαγραφή Βήματος"
   - Κουμπί "Προσθήκη Νέου Βήματος"

3. **Dialog Επεξεργασίας Βήματος**:
   - Entry για τίτλο
   - Text για περιγραφή
   - Spinbox για χρόνο
   - Listbox για υλικά βήματος με δυνατότητα προσθήκης/αφαίρεσης
